Erste Bank Client
================

Unofficial client for interacting with Erste Bank API

Currently supported only for Croatian bank

Supported APIs:
- [x] Accounts
- [x] Transactions

#### Installation via Composer
``` bash
$ composer require bornfight/erste-bank-client
```

#### Running tests
``` bash
$ composer test
```
